# Troubleshooting Exceptions: System.ServiceModel.Dispatcher.FilterInvalidBodyAccessException
A <xref:System.ServiceModel.Dispatcher.FilterInvalidBodyAccessException> exception is thrown when a filter or filter table attempts to access the body of an unbuffered message.
