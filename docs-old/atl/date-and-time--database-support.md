# Date and Time: Database Support
As of version 4.0, MFC database programming uses the [COleDateTime](../atl/coledatetime-class.md) and [COleDateTimeSpan](../atl/coledatetimespan-class.md) classes to represent date and time data. These classes, also used in Automation, are derived from class [COleVariant](../mfcref/colevariant-class.md). They supply better support for managing date and time data than do [CTime](../atl/ctime-class.md) and [CTimeSpan](../atl/ctimespan-class.md).
