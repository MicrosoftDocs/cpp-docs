# Cannot find .NET Framework directory: &lt;error&gt;
The directory where the [!INCLUDE[dnprdnshort](../ide/includes/dnprdnshort_md.md)] assemblies are stored could not be located.  
  
 **Error ID:** BC31508  
  
### To correct this error  
  
1.  Compile the program again to see if the error recurs.  
  
2.  If the error continues, save your work and restart [!INCLUDE[vsprvs](../build/includes/vsprvs_md.md)].  
  
3.  If the error recurs, reinstall [!INCLUDE[vbprvb](../cli/includes/vbprvb_md.md)].  
  
4.  If the error persists after reinstallation, notify Microsoft Product Support Services.
