# Method arguments must be enclosed in parentheses
The rules governing procedure calls are simpler in newer versions of [!INCLUDE[vbprvb](../cli/includes/vbprvb_md.md)]. All arguments must be enclosed by parentheses.  
  
 **Error ID:** BC30800  
  
### To correct this error  
  
-   Enclose the argument list in parentheses; for example:  
  
    ```  
    MySub(ArrayIndex, NewValue)  
    ```
