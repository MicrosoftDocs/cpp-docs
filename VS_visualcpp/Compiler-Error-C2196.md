# Compiler Error C2196
case value 'value' already used.  
  
 A switch statement uses the same case value more than once.  
  
 The following sample generates C2196:  
  
```  
// C2196.cpp  
int main() {  
   int i = 0;  
   switch( i ) {  
   case 0:  
      break;  
   case 0:   // C2196  
   // try the following line instead  
   // case 1:  
      break;  
   }  
}  
```
