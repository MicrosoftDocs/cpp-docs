# File Handling Classes
These classes provide methods for handling files, temporary files, and memory-mapped files.  
  
-   [CAtlFile](../VS_visualcpp/CAtlFile-Class.md) This class provides a thin wrapper around the Windows file-handling API.  
  
-   [CAtlFileMapping](../VS_visualcpp/CAtlFileMapping-Class.md) This class represents a memory-mapped file, adding a cast operator to the methods of [CAtlFileMappingBase](../VS_visualcpp/CAtlFileMappingBase-Class.md).  
  
-   [CAtlFileMappingBase](../VS_visualcpp/CAtlFileMappingBase-Class.md) This class represents a memory-mapped file.  
  
-   [CAtlTemporaryFile](../VS_visualcpp/CAtlTemporaryFile-Class.md) This class provides methods for the creation and use of a temporary file.
