# Compiler Error C3902
'accessor': type of last parameter must be 'type'  
  
 The type of the last parameter of at least one set method must match the type of the property. For more information, see [property](../VS_visualcpp/property---C---Component-Extensions-.md).  
  
 The following sample generates C3902:  
  
```  
// C3902.cpp  
// compile with: /clr /c  
using namespace System;  
ref class X {  
   property String ^Name {  
      void set(int);   // C3902  
      // try the following line instead  
      // void set(String^){}  
   }  
  
   property double values[int,int] {  
      void set(int, int, float);   // C3902  
      // try the following line instead  
      // void set(int, int, double){}  
   }  
};  
```
