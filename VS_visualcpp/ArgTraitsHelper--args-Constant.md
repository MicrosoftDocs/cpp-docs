# ArgTraitsHelper::args Constant
Supports the WRL infrastructure and is not intended to be used directly from your code.  
  
## Syntax  
  
```  
static const int args = Traits::args;  
```  
  
## Remarks  
 Helps [ArgTraitsHelper::args](../VS_visualcpp/ArgTraitsHelper--args-Constant.md) keep count of the number of parameters on the Invoke method of a delegate interface.  
  
## Requirements  
 **Header:** event.h  
  
 **Namespace:** Microsoft::WRL::Details
