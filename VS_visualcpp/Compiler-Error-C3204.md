# Compiler Error C3204
'_alloca' cannot be called from within a catch block  
  
 This error occurs when you use a call to [_alloca](../VS_visualcpp/_alloca.md) from within a catch block.  
  
## Example  
 The following sample generates C3204:  
  
```  
// C3204.cpp  
// compile with: /EHsc  
#include <malloc.h>  
  
void ShowError(void)  
{  
   try  
   {  
   }  
   catch(...)  
   {  
      _alloca(1);   // C3204  
   }  
}  
```
