# Compiler Error C3839
cannot change alignment in a managed or WinRT type  
  
 Alignment of variables in managed or Windows Runtime types is controlled by the CLR or Windows Runtime and cannot be modified with [align](../VS_visualcpp/align--C---.md).  
  
 The following sample generates C3839:  
  
```  
// C3839a.cpp  
// compile with: /clr  
ref class C  
{  
public:  
   __declspec(align(32)) int m_j; // C3839  
};  
  
int main()  
{  
}  
```
