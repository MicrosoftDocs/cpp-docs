# Compiler Warning (level 1) C4997
'class': coclass does not implement a COM interface or pseudo-interface  
  
 A class marked with the [coclass](../VS_visualcpp/coclass.md) attribute did not implement an interface.  
  
 The following sample generates C4997:  
  
```  
// C4997.cpp  
// compile with: /WX  
// to resolve this C4997, uncomment all code  
#include <objbase.h>  
  
[ object ]  
__interface I {  
   HRESULT func();  
};  
  
[ coclass ]  
struct C /*: I*/ {  
   /*  
   HRESULT func() {  
      return S_OK;  
   }  
   */  
};   // C4997  
```
