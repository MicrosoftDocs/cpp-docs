# Targets
In a dependency line, specify one or more targets, using any valid filename, directory name, or [pseudotarget](../VS_visualcpp/Pseudotargets.md). Separate multiple targets with one or more spaces or tabs. Targets are not case sensitive. Paths are permitted with filenames. A target cannot exceed 256 characters. If the target preceding the colon is a single character, use a separating space; otherwise, NMAKE interprets the letter-colon combination as a drive specifier.
