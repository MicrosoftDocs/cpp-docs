# Container Class::const_iterator
> [!NOTE]
>  This topic is in the Visual C++ documentation as a nonfunctional example of containers used in the Standard C++ Library. For more information, see [STL Containers](../VS_visualcpp/STL-Containers.md).  
  
 Describes an object that can serve as a constant iterator for the controlled sequence.  
  
## Syntax  
  
```  
  
typedef T6 const_iterator;  
  
```  
  
## Remarks  
 It is described here as a synonym for the unspecified type **T6**.
