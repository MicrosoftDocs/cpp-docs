# &lt;cvt-wbuffer&gt;
The header `<cvt/wstring>` in previous versions of Visual Studio defined the template class [wbuffer_convert Class](../VS_visualcpp/wbuffer_convert-Class.md) in the stdext::cvt namespace. The header is maintained for backward compatibility. New code should use the version of the class that is defined in [<locale\>](../VS_visualcpp/-locale-.md) in the `std` namespace  
  
## Syntax  
  
```  
#include <cvt/wbuffer>  
  
```
