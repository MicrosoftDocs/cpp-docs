# Compiler Error C3466
'type' : a specialization of a generic class cannot be forwarded  
  
 You cannot use type forwarding on a specialization of a generic class.  
  
 For more information, see [Type Forwarding (C++/CLI)](../VS_visualcpp/Type-Forwarding--C---CLI-.md).  
  
## Example  
 The following sample creates a component.  
  
```  
// C3466.cpp  
// compile with: /clr /LD  
generic<typename T>  
public ref class GR {};  
  
public ref class GR2 {};  
```  
  
## Example  
 The following sample generates C3466.  
  
```  
// C3466_b.cpp  
// compile with: /clr /c  
#using "C3466.dll"  
[assembly:TypeForwardedTo(GR<int>::typeid)];   // C3466  
[assembly:TypeForwardedTo(GR2::typeid)];   // OK  
```
