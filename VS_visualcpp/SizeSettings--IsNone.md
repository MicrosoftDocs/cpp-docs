# SizeSettings::IsNone
Checks if the resize data specifies no resizing.  
  
## Syntax  
  
```vb  
BOOL IsNone() const  
```  
  
## Return Value  
 TRUE if the `SizeSettings` object specifies no resizing.

## Requirements
