# Using a Dialog Template in Memory
Instead of using the methods given in the [Dialog Creation table](../VS_visualcpp/Creating-a-Dialog-Class-with-Code-Wizards.md), you can create either kind of dialog box indirectly from a dialog template in memory. For more information, see class [CDialog](../VS_visualcpp/CDialog-Class.md) in the *MFC Reference*.
