# &lt;typeinfo&gt;
Include the standard header `<typeinfo>` to define several types associated with the type-identification operator [typeid Operator](../VS_visualcpp/typeid-Operator.md), which yields information about both static and dynamic types.  
  
## Syntax  
  
```  
#include <typeinfo>  
  
```  
  
## Remarks  
 For information on classes defined in <typeinfo\>, see the following topics:  
  
-   [bad_cast Exception](../VS_visualcpp/bad_cast-Exception.md)  
  
-   [bad_typeid Exception](../VS_visualcpp/bad_typeid-Exception.md)  
  
-   [type_info Class](../VS_visualcpp/type_info-Class.md)
