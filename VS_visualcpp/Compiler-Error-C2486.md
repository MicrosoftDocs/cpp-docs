# Compiler Error C2486
'__LOCAL_SIZE' only allowed in function with the 'naked' attribute  
  
 In inline assembly functions, the name `__LOCAL_SIZE` is reserved for functions declared with the [naked](../VS_visualcpp/naked--C---.md) attribute.  
  
 The following sample generates C2486:  
  
```  
// C2486.cpp  
// processor: x86  
void __declspec(naked) f1() {  
   __asm {  
      mov   eax,   __LOCAL_SIZE  
   }  
}  
void f2() {  
   __asm {  
      mov   eax,   __LOCAL_SIZE   // C2486  
   }  
}  
```
