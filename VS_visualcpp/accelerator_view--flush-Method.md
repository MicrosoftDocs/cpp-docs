# accelerator_view::flush Method
Submits all pending commands queued to the [accelerator_view](../VS_visualcpp/accelerator_view-Class.md) object to the accelerator for execution.  
  
## Syntax  
  
```  
void flush();  
```  
  
## Return Value  
 Returns `void`.  
  
## Requirements  
 **Header:** amprt.h  
  
 **Namespace:** Concurrency
