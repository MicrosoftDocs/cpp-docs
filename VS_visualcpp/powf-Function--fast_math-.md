# powf Function (fast_math)
Calculates _X raised to the power of _Y  
  
## Syntax  
  
```  
inline float powf(  
    float _X,  
    float _Y ) restrict(amp);  
```  
  
#### Parameters  
 `_X`  
 Floating-point value, base  
  
 `_Y`  
 Floating-point value, exponent  
  
## Return Value  
  
## Requirements  
 **Header:** amp_math.h  
  
 **Namespace:** Concurrency::fast_math
