##  <a name="istringstream"></a>  istringstream  
 Creates a type `basic_istringstream` specialized on a `char` template parameter.  
  
```  
typedef basic _ istringstream<char> istringstream;  
  
```  
  
### Remarks  
 The type is a synonym for template class [basic_istringstream](../VS_visualcpp/basic_istringstream-Class.md), specialized for elements of type `char`*.*
