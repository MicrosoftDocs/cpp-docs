# Compiler Warning (level 1) C4075
initializers put in unrecognized initialization area  
  
 A [#pragma init_seg](../VS_visualcpp/init_seg.md) uses an unrecognized section name. The compiler ignores the **pragma** command.  
  
 The following sample generates C4075:  
  
```  
// C4075.cpp  
// compile with: /W1  
#pragma init_seg("mysegg")   // C4075  
  
// try..  
// #pragma init_seg(user)  
  
int main() {  
}  
```
