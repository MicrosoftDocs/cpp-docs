# CAccessorRowset::Close
Releases any active accessors and the rowset.  
  
## Syntax  
  
```  
  
void Close( );  
  
```  
  
## Remarks  
 Releases any associated memory.  
  
## Requirements  
 **Header:** atldbcli.h
