# accelerator_view::is_debug Data Member
Gets a Boolean value that indicates whether the [accelerator_view](../VS_visualcpp/accelerator_view-Class.md) object has the  DEBUG layer enabled for extensive error reporting.  
  
## Syntax  
  
```  
__declspec(property(get= get_is_debug)) bool is_debug;  
```  
  
## Requirements  
 **Header:** amprt.h  
  
 **Namespace:** Concurrency
