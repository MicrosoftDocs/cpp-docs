# accelerator_view::get_is_debug Method
Returns a Boolean value that indicates whether the [accelerator_view](../VS_visualcpp/accelerator_view-Class.md) object has the DEBUG layer enabled for extensive error reporting.  
  
## Syntax  
  
```  
bool get_is_debug() const;  
```  
  
## Return Value  
 A Boolean value that indicates whether the `accelerator_view` object has the DEBUG layer enabled for extensive error reporting.  
  
## Requirements  
 **Header:** amprt.h  
  
 **Namespace:** Concurrency
