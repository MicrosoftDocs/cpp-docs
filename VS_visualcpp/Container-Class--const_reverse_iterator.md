# Container Class::const_reverse_iterator
> [!NOTE]
>  This topic is in the Visual C++ documentation as a nonfunctional example of containers used in the Standard C++ Library. For more information, see [STL Containers](../VS_visualcpp/STL-Containers.md).  
  
 Describes an object that can serve as a constant reverse iterator for the controlled sequence.  
  
## Syntax  
  
```  
  
typedef T8 const_reverse_iterator;  
  
```  
  
## Remarks  
 It is described here as a synonym for the unspecified type **T8** (typically [reverse_iterator](../VS_visualcpp/Container-Class--reverse_iterator.md) <[const_iterator](../VS_visualcpp/Container-Class--const_iterator.md)`>`).
