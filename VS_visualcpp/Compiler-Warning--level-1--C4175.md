# Compiler Warning (level 1) C4175
\#pragma component(browser, on) : browser info must initially be specified on the command line  
  
 To use [component](../VS_visualcpp/component.md) pragma, you must generate browse information during compilation ([/FR](../VS_visualcpp/-FR---Fr--Create-.Sbr-File-.md)).
