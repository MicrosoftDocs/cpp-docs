# Compiler Warning (level 4) C4536
'type name' : type-name exceeds meta-data limit of 'limit' characters  
  
 A type name would be truncated in metadata if it was a managed type. See [C3180](../VS_visualcpp/Compiler-Error-C3180.md) for more information.  
  
 This warning is off by default. See [Compiler Warnings That Are Off by Default](../VS_visualcpp/Compiler-Warnings-That-Are-Off-by-Default.md) for more information.
