# .ERRE
[!INCLUDE[blank_token](../../includes/blank-token.md)]

The latest version of this topic can be found at [.ERRE](https://docs.microsoft.com/cpp/assembler/masm/dot-erre).  
  
  
Generates an error if *expression* is false (0).  
  
## Syntax  
  
```  
  
.ERRE expression [[, message]]  
```
