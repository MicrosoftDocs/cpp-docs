# Resource Compiler Error RC2145
PRIMARY LANGUAGE ID too large  
  
 The PRIMARY LANGUAGE ID must be a valid hexadecimal language ID. See [Language and Country/Region Strings](../c-runtime-library/locale-names-languages-and-country-region-strings.md) in the *Run-Time Library Reference* for a list of valid Language IDs.
