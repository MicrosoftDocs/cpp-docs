# CArrayRowset Class

## Syntax  
  
```

class CArrayRowset :   
   public CVirtualBuffer <TAccessor>,Â   
   protected CBulkRowset <TAccessor>  
```  
  
#### Parameters  
 `TAccessor`  
 The type of accessor class that you want the rowset to use.
