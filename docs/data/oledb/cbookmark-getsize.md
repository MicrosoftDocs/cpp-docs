# CBookmark::GetSize
Retrieves the size of the bookmark buffer.  
  
## Syntax  
  
```  
  
virtual DBLENGTH GetSize( ) const throw( );  
  
```  
  
## Return Value  
 The size of the buffer in bytes.  
  
## Requirements  
 **Header:** atldbcli.h
