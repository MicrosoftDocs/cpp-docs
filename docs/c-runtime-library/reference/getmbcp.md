# _getmbcp

Retrieves the current code page.

## Syntax

```C
int _getmbcp( void );
```

## Return Value

Returns the current multibyte code page. A return value of 0 indicates that a single byte code page is in use.

## Requirements

|Routine|Required header|
|-------------|---------------------|
|**_getmbcp**|\<mbctype.h>|
