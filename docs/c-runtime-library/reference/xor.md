# xor
An alternative to the ^ operator.  
  
## Syntax  
  
```  
  
#define xor ^  
  
```  
  
## Remarks  
 The macro yields the operator ^.  
  
## Example  
  
```  
// iso646_xor.cpp  
// compile with: /EHsc  
#include <iostream>  
#include <iso646.h>  
  
int main( )  
{  
   using namespace std;  
   int a = 3, b = 2, result;  
  
   result= a ^ b;  
   cout << result << endl;  
  
   result= a xor_eq b;  
   cout << result << endl;  
}  
```  
  
```Output  
1  
1  
```  
  
## Requirements  
 **Header:** \<iso646.h>
