# partition (STL/CLR)
Classifies elements in a range into two disjoint sets, with those elements satisfying a unary predicate preceding those that fail to satisfy it.  
  
## Syntax  
  
```  
template<class _BidIt, class _Pr> inline  
    _BidIt partition(_BidIt _First, _BidIt _Last, _Pr _Pred);  
```  
  
## Remarks  
 This function behaves the same as the STL function `partition`. For more information, see [partition](http://msdn.microsoft.com/Library/01a91b09-7950-4f5e-921d-3b2457ed5be4).  
  
## Requirements  
 **Header:** \<cliext/algorithm>  
  
 **Namespace:** cliext
