# Navigating the Class Structure (Visual C++)

You can navigate a project class structure, including functions, inheritance and interfaces using either **Class View** or the **Object Browser**. For more information, see [Viewing the structure of code](/visualstudio/ide/viewing-the-structure-of-code).

You can go to a function declaration by double-clicking on the function name in Class View. You can go to a class definition by double-clicking on a class name in Class View.
