# __addfsbyte, __addfsword, __addfsdword

**Microsoft Specific**

Add a value to a memory location specified by an offset relative to the beginning of the `FS` segment.

## Syntax

```
void __addfsbyte(
   unsigned long Offset,
   unsigned char Data
);
void __addfsword(
   unsigned long Offset,
   unsigned short Data
);
void __addfsdword(
   unsigned long Offset,
   unsigned long Data
);
```

#### Parameters

*Offset*<br/>
[in] The offset from the beginning of `FS`.

*Data*<br/>
[in] The value to add to the memory location.

## Requirements

|Intrinsic|Architecture|
|---------------|------------------|
|`__addfsbyte`|x86|
|`__addfsword`|x86|
|`__addfsdword`|x86|

## Remarks

These routines are available only as intrinsics.

**END Microsoft Specific**
