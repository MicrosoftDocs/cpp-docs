# Resource Compiler Fatal Error RC1020
unexpected '#endif'  
  
 An `#endif` directive appeared without a matching `#if`, **#ifdef**, or **#ifndef** directive.  
  
 Make sure that there is a matching `#endif` for each `#if`, **#ifdef**, and **#ifndef** statement.
