# Resource Compiler Fatal Error RC1019

unexpected '#else'

The `#else` directive did not appear within an `#if`, **#ifdef**, or **#ifndef** construct.

Make sure that there is an `#if`, **#ifdef**, or **#ifndef** statement in effect before this statement.
