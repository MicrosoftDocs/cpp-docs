# Expression Evaluator Error CXX0045
not a function  
  
 An argument list was supplied for a symbol in the program that is not the name of a function.  
  
## Example  
  
```  
queue( alpha, beta )  
```  
  
 when `queue` is not a function.  
  
 This error is identical to CAN0045.
