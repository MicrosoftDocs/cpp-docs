# Expression Evaluator Error CXX0018

> bad register name

## Remarks

A specified register does not exist or cannot be displayed.

The Watch window can display the following registers:

AX    SP    DS    IP

BX    BP    ES    FL

CX    SI    SS    GS

DX    DI    CS    SS

EAX    ESP    DS    EIP

EBX    EBP    ES    EFL

ECX    ESI    FS

EDX    EDI    CS

This error is identical to CAN0018.
