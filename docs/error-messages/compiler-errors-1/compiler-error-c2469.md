# Compiler Error C2469

> '`new`': cannot allocate '`void`' objects

An operator was passed an invalid type.

The following sample generates C2469:

```cpp
// C2469.cpp
int main() {
   int *i = new void;   // C2469
   int *i = new int;   // OK
}
```
