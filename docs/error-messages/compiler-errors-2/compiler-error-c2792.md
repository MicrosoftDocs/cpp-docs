# Compiler Error C2792
'super' : this keyword must be followed by '::'  
  
 The only token that can follow the keyword `__super` is `::`.  
  
 The following sample generates C2792:  
  
```  
// C2792.cpp  
struct B {  
   void mf();  
};  
  
struct D : B {  
   void mf() {  
      __super.();   // C2792  
  
      // try the following line instead  
      // __super::mf();  
   }  
};  
```
