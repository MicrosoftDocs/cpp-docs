# ActiveX Controls

In Visual C++ you can create ActiveX controls using MFC or ATL.

>[!IMPORTANT]
> ActiveX is a legacy technology that should not be used for new development. Many capabilities of ActiveX controls can be performed in a simpler and much more secure way with modern technologies such as HTML5 and JavaScript, modern browser extensions, or WebAssembly modules.
