# _com_ptr_t::GetInterfacePtr

**Microsoft Specific**

Returns the encapsulated interface pointer.

## Syntax

```
Interface* GetInterfacePtr( ) const throw( );
Interface*& GetInterfacePtr() throw();
```

## Remarks

Returns the encapsulated interface pointer, which may be NULL.
