# EXITM
Terminates expansion of the current repeat or macro block and begins assembly of the next statement outside the block.  
  
## Syntax  
  
```  
  
EXITM [[textitem]]   
```  
  
## Remarks  
 In a macro function, *textitem* is the value returned.
