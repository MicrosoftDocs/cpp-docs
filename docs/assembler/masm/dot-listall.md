# .LISTALL
Starts listing of all statements.  
  
## Syntax  
  
```  
  
.LISTALL  
  
```  
  
## Remarks  
 Equivalent to the combination of [.LIST](../../assembler/masm/dot-list.md), [.LISTIF](../../assembler/masm/dot-listif.md), and [.LISTMACROALL](../../assembler/masm/dot-listmacroall.md).
