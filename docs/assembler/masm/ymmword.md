# YMMWORD

Used for 256-bit multimedia operands with Intel Advanced Vector Extensions (AVX) instructions.

## Syntax

> YMMWORD

## Remarks

`YMMWORD` is intended to represent the same type as `__m256` for the AVX intrinsics.
