# .CONTINUE

Generates code to jump to the top of a [.WHILE](../../assembler/masm/dot-while.md) or [.REPEAT](../../assembler/masm/dot-repeat.md) block if `condition` is true.

## Syntax

> .CONTINUE [[.IF condition]]
