# PUBLIC (MASM)

Makes each variable, label, or absolute symbol specified as *name* available to all other modules in the program.

## Syntax

> **PUBLIC** ⟦language-type⟧ *name* ⟦__,__ ⟦language-type⟧ *name* ...⟧
