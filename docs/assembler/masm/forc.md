# FORC

Marks a block that will be repeated once for each character in *string*, with the current character replacing *parameter* on each repetition.

## Syntax

> FORC *parameter*, \<*string*><br/>
> *statements*<br/>
> ENDM

## Remarks

Same as [IRPC](../../assembler/masm/irpc.md).
