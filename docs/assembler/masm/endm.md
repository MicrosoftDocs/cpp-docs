# ENDM

Terminates a macro or repeat block.

## Syntax

> ENDM

## Remarks

See [MACRO](../../assembler/masm/macro.md), [FOR](../../assembler/masm/for-masm.md), [FORC](../../assembler/masm/forc.md), [REPEAT](../../assembler/masm/repeat.md), or **WHILE**.
