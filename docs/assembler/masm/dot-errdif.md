# .ERRDIF]]
Generates an error if the text items are different.  
  
## Syntax  
  
```  
  
.ERRDIF[[I]] <textitem1>, <textitem2> [[, message]]  
```  
  
## Remarks  
 If **I** is given, the comparison is case insensitive.
