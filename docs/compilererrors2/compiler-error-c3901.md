# Compiler Error C3901
'accessor_function': must have return type 'type'  
  
 At least one get method's return type must match the property type. For more information, see [property](../windows/property---c---component-extensions-.md).  
  
 The following sample generates C3901:  
  
```  
// C3901.cpp  
// compile with: /clr /c  
using namespace System;  
ref class X {  
   property String^ Name {  
      void get();   // C3901  
      // try the following line instead  
      // String^ get();  
   };  
};  
  
ref class Y {  
   property double values[int, int] {  
      int get(int, int);   // C3901  
      // try the following line instead  
      // double get(int, int);  
   };  
};  
```
