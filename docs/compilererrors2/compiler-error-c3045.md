# Compiler Error C3045
Expected a compound statement following OpenMP 'sections' directive. Missing '{'  
  
 A code block delimited by braces must follow a [sections](../parallel/sections--openmp-.md) directive.  
  
 The following sample generates C3045:  
  
```  
// C3045.cpp  
// compile with: /openmp /c  
#include "omp.h"  
  
int main() {  
   int n2 = 2, n3 = 3;  
  
   #pragma omp parallel  
   {  
      ++n2;  
  
      #pragma omp sections  
         ++n2;   // C3045  
  
      #pragma omp sections   // OK  
      {  
         ++n3;  
      }  
   }  
}  
```
