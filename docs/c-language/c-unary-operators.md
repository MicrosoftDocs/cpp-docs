# C Unary Operators

Unary operators appear before their operand and associate from right to left.

## Syntax

*unary-expression*:
*postfix-expression*

**++**  *unary-expression*

`--`  *unary-expression*

*unary-operator cast-expression*

**`sizeof`**  *unary-expression*

**sizeof (**  *type-name*  **)**

*unary-operator*: one of
**& \* + -** `~` **!**
