# named_guids
**C++ Specific**  
  
Tells the compiler to define and initialize GUID variables in old style, of the form `LIBID_MyLib`, `CLSID_MyCoClass`, `IID_MyInterface`, and `DIID_MyDispInterface`.  
  
## Syntax  
  
```  
named_guids  
```
