# CLS Compliance Warnings
The topics in this section are for the warnings that can be generated by the linker when performing Common Language Subset (CLS) compliance checks on your assembly.  
  
 For more information, see [CLS Compliant Assemblies](http://msdn.microsoft.com/en-us/3320b57e-ea55-4697-a17d-f509a36a3c93).
