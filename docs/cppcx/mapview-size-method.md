# MapView::Size Method
Returns the number of elements in the current MapView object.  
  
## Syntax  
  
```  
  
virtual property unsigned int Size;  
```  
  
## Return Value  
 The number of elements in the current MapView.  
  
## Requirements  
 **Header:** collection.h  
  
 **Namespace:** Platform::Collections
