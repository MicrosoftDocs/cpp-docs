# A failure occurred while attempting to start the compilation
An unknown error has been encountered while issuing the **Start Build** request to the [!INCLUDE[vbprvb](../cli/includes/vbprvb_md.md)] or [!INCLUDE[csprcs](../ide/includes/csprcs_md.md)] compiler.  
  
 **To correct this error**  
  
-   Restart [!INCLUDE[vsprvs](../build/includes/vsprvs_md.md)].  
  
     This error will cause the build process to fail.
