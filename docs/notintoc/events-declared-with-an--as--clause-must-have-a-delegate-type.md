# Events declared with an &#39;As&#39; clause must have a delegate type
A type was specified that is not a delegate.  
  
 **Error ID:** BC31044  
  
### To correct this error  
  
1.  Remove the `As` clause.  
  
2.  Specify a delegate with the `As` clause.
