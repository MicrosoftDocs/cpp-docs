# SRWLockSharedTraits::GetInvalidValue Method

Retrieves an **SRWLockSharedTraits** object that is always invalid.

## Syntax

```cpp
inline static Type GetInvalidValue();
```

## Return Value

A handle to a **SRWLockSharedTraits** object.
