# Nil Structure

Supports the WRL infrastructure and is not intended to be used directly from your code.

## Syntax

```cpp  
struct Nil;  
```

## Remarks

Used to indicate an unspecified, optional template parameter.

**Nil** is an empty structure.

`Nil`
