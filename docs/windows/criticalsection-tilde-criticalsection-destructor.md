# CriticalSection::~CriticalSection Destructor

Deinitializes and destroys the current **CriticalSection** object.

## Syntax

```cpp
WRL_NOTHROW ~CriticalSection();
```

## Requirements

**Header:** corewrappers.h

**Namespace:** Microsoft::WRL::Wrappers
