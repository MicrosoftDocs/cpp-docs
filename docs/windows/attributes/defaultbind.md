# defaultbind

Indicates the single, bindable property that best represents the object.

## Syntax

```cpp
[defaultbind]
```

## Remarks

The **defaultbind** C++ attribute has the same functionality as the [defaultbind](/windows/win32/Midl/defaultbind) MIDL attribute.

## Example

See the example for [bindable](bindable.md) for an example of how to use **defaultbind**.

## Requirements

| Attribute context | Value |
|-|-|
|**Applies to**|Interface method|
|**Repeatable**|No|
|**Required attributes**|None|
|**Invalid attributes**|None|

For more information, see [Attribute Contexts](cpp-attributes-com-net.md#contexts).
