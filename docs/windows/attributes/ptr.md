# ptr

Designates a pointer as a full pointer.

## Syntax

```cpp
[ptr]
```

## Remarks

The **ptr** C++ attribute has the same functionality as the [ptr](/windows/desktop/Midl/ptr) MIDL attribute.

## Example

See the example for [defaultvalue](defaultvalue.md) for a sample use of **ptr**.

## Requirements

### Attribute Context

|||
|-|-|
|**Applies to**|Interface parameter, interface method, **typedef**|
|**Repeatable**|No|
|**Required attributes**|None|
|**Invalid attributes**|None|

For more information about the attribute contexts, see [Attribute Contexts](cpp-attributes-com-net.md#contexts).
