# helpstringcontext

Specifies the ID of a help topic in an .hlp or .chm file.

## Syntax

```cpp
[ helpstringcontext(
   contextID
) ]
```

### Parameters

*contextID*  
A 32-bit Help context identifier in the **Help** file.

## Remarks

The **helpstringcontext** C++ attribute has the same functionality as the [helpstringcontext](/windows/desktop/Midl/helpstringcontext) ODL attribute.

## Example

```cpp
// cpp_attr_ref_helpstringcontext.cpp
// compile with: /LD
#include <unknwn.h>
[module(name="MyLib")];

[   object,
   helpstring("help string"),
   helpstringcontext(1),
   uuid="11111111-1111-1111-1111-111111111111"
]
__interface IMyI
{
   HRESULT xx();
};
```
