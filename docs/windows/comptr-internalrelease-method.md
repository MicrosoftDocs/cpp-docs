# ComPtr::InternalRelease Method

Performs a COM Release operation on the interface associated with this **ComPtr**.

## Syntax

```cpp  
void InternalRelease();  
```

## Remarks

This method is protected.

## Requirements

**Header:** client.h

**Namespace:** Microsoft::WRL
