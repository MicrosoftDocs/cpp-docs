# ComPtr::Reset

Releases all references for the pointer to the interface that is associated with this **ComPtr**.

## Syntax

```cpp
unsigned long Reset();
```

## Return Value

The number of references released, if any.

## Requirements

**Header:** client.h

**Namespace:** Microsoft::WRL
