# displaybind

Indicates a property that should be displayed to the user as bindable.

## Syntax

```cpp
[displaybind]
```

## Remarks

The **displaybind** C++ attribute has the same functionality as the [displaybind](https://msdn.microsoft.com/library/windows/desktop/aa366804) MIDL attribute.

## Example

See the example for [bindable](../windows/bindable.md) for an example of how to use **displaybind**.
