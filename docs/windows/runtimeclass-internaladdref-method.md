# RuntimeClass::InternalAddRef Method

Increments the reference count to the current **RuntimeClass** object.

## Syntax

```cpp
ULONG InternalAddRef();
```

## Return Value

The resulting reference count.

## Requirements

**Header:** implements.h

**Namespace:** Microsoft::WRL
