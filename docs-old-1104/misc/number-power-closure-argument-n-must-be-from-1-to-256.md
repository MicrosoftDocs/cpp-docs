# &lt;number&gt; power closure argument &#39;n&#39; must be from 1 to 256.
This error generally occurs when the number exceeds the accepted range of 1 to 256.  
  
### To correct this error  
  
1.  Enter a number between 1 and 256 and try again. For more information on syntax, see [Regular Expressions](/visual-studio/ide/using-regular-expressions-in-visual-studio).
